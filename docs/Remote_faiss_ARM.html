遇到的问题 (6)

GitHub无法直接访问

方案: 本地克隆后上传到远程主机

来源: network connectivity issue

补丁应用行尾符不匹配（Windows CRLF vs Linux LF）

方案: 使用dos2unix转换行尾符

来源: file format issue

CMake版本要求3.23.1，但系统版本为3.22.0

方案: 修改CMakeLists.txt降低版本要求为3.22.0

来源: CMake version compatibility

GLIBCXX版本不匹配（faiss用GCC11编译，系统为GCC10）

方案: 使用系统GCC 10.3.1重新编译faiss

来源: ABI compatibility

系统未安装OpenBLAS包

方案: yum install openblas openblas-devel

来源: dependency

测试程序构建脚本需要交互式输入EXTRA_DEFINES

方案: 通过echo '' | make提供空输入

来源: build script interaction
