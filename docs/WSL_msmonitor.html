文档缺失 (5)

#1install_guide.md 未列出 unzip 依赖（构建插件时 cmake 自动下载 sqlite3.zip 需要 unzip 解压）
#2install_guide.md 的 Rust 安装使用 rustup（curl 下载），openEuler dnf 仓库直接有 rust-1.82.0 更便捷
#3dynolog 完整构建依赖 dynolog.tar.gz（华为云OBS），网络不可达时会阻塞构建
#4whl 运行时依赖 libmspti.so（Ascend msPTI），文档未在安装前提中明确标注此运行时依赖
#5plugin/build.sh 自动从 gitee/华为云OBS 下载第三方依赖（glog/nlohmann-json/sqlite3），首次构建耗时长

遇到的问题 (4)

第1次构建失败：unzip: command not found（cmake 下载 sqlite3.zip 后无法解压）

方案: dnf install -y unzip（openEuler 仓库原生包）。文档 install_guide.md 未列出此依赖

来源: plugin/cmake/download_opensource.sh:98 行调用 unzip

误操作：rm -rf plugin/third_party 删除了 securec 头文件，导致 C++ 编译失败（securec.h: No such file）

方案: git checkout plugin/third_party/securec 恢复。securec 是仓库内文件不是下载依赖

来源: plugin/ipc_monitor/NpuIpcEndPoint.h:28 #include "securec.h"

冒烟测试 whl 导入失败：ImportError: libmspti.so: cannot open shared object file

方案: libmspti.so 是 Ascend msPTI（性能分析工具接口）的共享库，需要 CANN + NPU 驱动环境。测试中通过 plugin/stub 下的 stub 库提供 mock。不影响构建和 UT 验证

来源: IPCMonitor_C.so 链接依赖

pre-commit pylint/bandit/typos 因宿主机 Python 3.9.9 和网络限制跳过

方案: 跳过，记录为环境限制

来源: .pre-commit-config.yaml
