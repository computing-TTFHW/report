遇到的问题 (2)

bingo build --stage=dev 首次执行失败，缺少 profile.luajit.ini 配置文件

方案: 通过 bingo build -t install_sdk -b openUBMC -ucc 安装 SDK 和 RTOS 编译工具链，自动生成 profile.luajit.ini/profile.ini/profile.dt.ini

来源: bingo 错误提示: Profile文件/root/.conan2/profiles/profile.luajit.ini 不存在

bingo build --stage=dev -o test=True 编译失败，test_utilities/test_base.h 头文件未找到

方案: 去掉 test 选项执行默认构建成功，随后通过 bingo test -ut -jit 单独执行单元测试成功

来源: conanfile.py 中 test 选项开启后 Conan 构建路径缺少 test_utilities 头文件
